# ADR-0002: State management

- Status: Proposed
- Date: 2026-07-22

## Context

SCE needs predictable asynchronous state for Google authentication, file selection,
parsing, simulation, synchronization, and dashboards.

## Proposed decision

Use Riverpod for state management and dependency wiring.

## Confirmation required

This ADR remains proposed until the team confirms the package choice and version.
